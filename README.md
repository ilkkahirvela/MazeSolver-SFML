# Maze Solver Visualization

A C++ project using **SFML** to generate and solve mazes with a step-by-step animated visualization.

## Features
- Maze generation with **recursive backtracking**.
- Maze solving with **Breadth-First Search (BFS)**.
- Animated visualization:
  - **Blue** cells = visited during search.
  - **Red** cells = final shortest path.
- Efficient rendering with static/dynamic layers for smooth performance.
