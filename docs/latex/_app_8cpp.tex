\doxysubsection{Detailed Description}
\doxylink{class_maze}{Maze} generation and solver visualization using SFML. 

This program generates a maze, solves it using BFS in a separate thread, and visualizes the solving process in real time.

Features\+:
\begin{DoxyItemize}
\item Static rendering of maze walls (drawn once into a render texture).
\item Dynamic rendering of visited cells (light blue) and final path (red).
\item Step-\/by-\/step animated visualization. 
\end{DoxyItemize}
