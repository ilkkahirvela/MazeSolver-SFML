\hypertarget{index_intro_sec}{}\doxysection{\texorpdfstring{Introduction}{Introduction}}\label{index_intro_sec}
This project demonstrates maze generation and solving with an animated visualization built using C++ and the SFML graphics library.\hypertarget{index_algo_sec}{}\doxysection{\texorpdfstring{Algorithms}{Algorithms}}\label{index_algo_sec}

\begin{DoxyItemize}
\item \doxylink{class_maze}{Maze} generation\+: Recursive Backtracking (Depth-\/\+First Search)
\item \doxylink{class_maze}{Maze} solving\+: Breadth-\/\+First Search (BFS)
\end{DoxyItemize}\hypertarget{index_viz_sec}{}\doxysection{\texorpdfstring{Visualization}{Visualization}}\label{index_viz_sec}

\begin{DoxyItemize}
\item \doxylink{class_maze}{Maze} walls are drawn once into a static layer.
\item Visited cells are shown in light blue as the solver explores.
\item The final shortest path is shown in red once discovered.
\end{DoxyItemize}\hypertarget{index_deps_sec}{}\doxysection{\texorpdfstring{Dependencies}{Dependencies}}\label{index_deps_sec}

\begin{DoxyItemize}
\item C++17 or newer
\item SFML 3.\+0+ (graphics, window, system)
\end{DoxyItemize}\hypertarget{index_usage_sec}{}\doxysection{\texorpdfstring{Usage}{Usage}}\label{index_usage_sec}
Build and run the program. An SFML window opens to show the maze generation and solving process step by step.
