\doxysubsection{Detailed Description}
Solves a generated maze using Breadth-\/\+First Search (BFS). 

The solver runs BFS in a background thread. During execution it records both visited cells (for visualization) and the final shortest path (if found). Thread-\/safety is ensured with mutexes and atomics.
