\doxysubsection{Detailed Description}
Represents a 2D maze grid and handles generation and rendering. 

The \doxylink{class_maze}{Maze} consists of a grid of \doxylink{class_cell}{Cell} objects. A recursive backtracker (depth-\/first search) algorithm is used to generate the maze structure.
