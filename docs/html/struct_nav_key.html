Detailed Description

Tracks a single key's press and hold state for velocity-based UI navigation.

On the first frame the key is pressed, justPressed is true and can be used to fire a single minimum step. On subsequent frames while held, heldSecs() drives a velocity ramp for smooth continuous movement.
