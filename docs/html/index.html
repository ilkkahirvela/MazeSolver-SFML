Maze Solver Visualization

Introduction

This project demonstrates maze generation and solving with an animated visualization built using C++ and the SFML graphics library.

Algorithms

  • Maze generation: Recursive Backtracking (Depth-First Search)
  • Maze solving: Breadth-First Search (BFS)

Visualization

  • Maze walls are drawn once into a static layer.
  • Visited cells are shown in light blue as the solver explores.
  • The final shortest path is shown in red once discovered.

Dependencies

  • C++17 or newer
  • SFML 3.0+ (graphics, window, system)

Usage

Build and run the program. An SFML window opens to show the maze generation and solving process step by step.
